# petshop

Projeto do site PetShow do módulo Node + TypeScript do curso da B7Web

### Pré-requisitos globais

`npm i -g nodemon typescript ts-node`

### Instalação

`git clone https://github.com/pierredcrodri/petshop.git`
`cd petshop`
`npm i`

### Inicialização

`npm run start-dev`
ou
`npm run start`
